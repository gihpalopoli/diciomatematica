O que é Estatística?

A Estatística é a área da matemática que se ocupa da coleta, organização, análise, interpretação e apresentação de dados. Seu objetivo é tirar conclusões a partir de dados coletados e usá-los para fazer previsões ou tomar decisões.

Principais Conceitos de Estatística

  • População: O conjunto completo de elementos que têm uma característica comum que se quer estudar.
  • Amostra: Uma parte representativa da população, escolhida para análise, quando não é possível analisar toda a população.
  • Variáveis: Características ou propriedades que podem ser medidas e variam entre os membros de uma população ou amostra.
  • Frequência: O número de vezes que um determinado valor ou categoria aparece em um conjunto de dados.

Distribuição de Frequências

Uma distribuição de frequências é uma tabela que organiza os dados para mostrar quantas vezes cada valor ocorre. Pode ser representada de forma simples, por exemplo:

Valor Frequência
2 5
4 8
6 3
8 7

Medidas de Tendência Central

As medidas de tendência central ajudam a resumir um conjunto de dados, fornecendo um valor representativo. As principais medidas são:

  • Moda: O valor que mais aparece em um conjunto de dados.
  • Mediana: O valor que divide o conjunto de dados ao meio, ou seja, 50% dos valores são menores e 50% são maiores.
  • Média: A soma de todos os valores dividida pelo número de elementos no conjunto. Exemplo: Para os dados {2, 4, 6, 8}, a média é (2 + 4 + 6 + 8) / 4 = 5.

Desvio Padrão

O desvio padrão é uma medida da dispersão dos dados. Quanto maior o desvio padrão, mais espalhados estão os dados em relação à média. Ele é usado para entender a variabilidade dos dados.

Gráficos Estatísticos

Os gráficos são ferramentas essenciais para a visualização de dados estatísticos. Alguns dos gráficos mais comuns incluem:

  • Gráfico de Barras: Usado para comparar a frequência de diferentes categorias.
  • Gráfico de Pizza: Usado para mostrar a proporção de cada categoria em relação ao total.
  • Histograma: Usado para mostrar a distribuição de dados contínuos em intervalos.

Exemplo de Aplicação

Suponha que você tenha coletado os seguintes dados sobre as idades de um grupo de 10 pessoas:
{20, 22, 25, 30, 22, 24, 28, 30, 22, 26}.

A média seria: (20 + 22 + 25 + 30 + 22 + 24 + 28 + 30 + 22 + 26) / 10 = 25.9. O desvio padrão pode ser calculado para entender melhor a dispersão dos dados.
